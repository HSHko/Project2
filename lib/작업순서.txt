🔻firebase 설치.
시작: https://firebase.google.com/
참고: https://www.youtube.com/watch?v=m_u6P5k0vP0
참고: https://firebase.google.com/docs/functions/get-started
⏩vscode에서 firebase 명령어를 사용하려면, Windows PowerShell 관리자실행 ->
⚡Set-ExecutionPolicy RemoteSigned  // 권한변경
⚡Get-ExecutionPolicy               // 권한확인

⚡npm install -g firebase-tools
⏩바탕화면의 cmd에서..
⚡firebase login
⚡firebase login:ci  // 로그인후 Token이 표시됨
⚡firebase logout    // 로그아웃 하고 싶을 때
⏩firebase 패키지를 설치할 폴더로 가서..
⚡firebase init
⏩코드 수정후 테스트할때...
⚡firebase emulators:start
⏩코드 수정후 본방 입력할때...
⚡firebase deploy
